# Double-Hand Ghost
This mod makes it look like you had a totem in your hand and offhand, as soon as you are in the deathscreen.

configuration:

```/toggleautoghost``` - toggle it on or off

```/autoghost <slot>``` - set the slot for the totem to appear in (preferably your main pvp totem slot)

# Build from source

```sh
git clone https://github.com/TheVoidBlock/doublehand-ghost
cd doublehand-ghost
./gradlew build
cd build/libs
```
